O Comando de Atribuição (=)

Na matemática, o sinal de igual (=) significa que os dois lados têm o mesmo valor. Na linguagem C, o igual significa "RECEBE".

A Regra de Ouro da Atribuição:

"O computador resolve TUDO o que está à DIREITA primeiro, e só no final guarda o resultado na variável que está à ESQUERDA."

int pontos = 10;


// Em matemática, a linha abaixo seria impossível.


// Em C, ela significa: "Pegue os pontos atuais (10),


// some 5, e guarde o resultado (15) de volta na caixa pontos".


pontos = pontos + 5;

Operadores Básicos

Os três primeiros operadores são exatamente iguais aos que aprendemos na escola.

Símbolo Operação Exemplo no Código
+ Adição soma = a + b;
- Subtração sub = 10 - 4;
* Multiplicação mult = 4 * 5;

Nota: O asterisco (*) substitui o clássico "x" ou "·" para evitar confusão com letras.

A Pegadinha da Divisão ( / )

A barra (/) faz a divisão, mas a linguagem C tem uma regra muito estrita sobre os tipos de dados que estão a ser divididos.

Divisão de Inteiros

Se você divide dois int, o C corta as casas decimais. Ele não arredonda, ele simplesmente joga a vírgula fora!

10 / 4 -> Resultado: 2

Divisão Real

Para obter um resultado decimal, pelo menos um dos números na conta precisa ser do tipo float ou double (ter ponto).

10.0 / 4.0 -> Resultado: 2.5

A Estrela da Aula: O Módulo ( % )

O símbolo % não significa percentagem em programação! Ele significa "O RESTO DA DIVISÃO INTEIRA".

10 / 3

A divisão dá 3

(Pois 3 x 3 = 9)

🤔

10 % 3

O Resto é 1

(10 menos 9 = 1)

CUIDADO MÁXIMO: O operador Módulo (%) SÓ FUNCIONA com variáveis do tipo inteiro (int).

Para que serve o Módulo?

Saber o que sobra de uma divisão é um dos truques mais usados na programação lógica.

1. Descobrir se é PAR ou ÍMPAR

Qualquer número par dividido por 2 não tem resto (sobra 0). Se sobrar 1, é ímpar!

se (numero % 2 == 0) -> PAR

2. Encontrar Múltiplos

Para saber se um número é divisível por outro (ex: anos bissextos são divisíveis por 4).

se (ano % 4 == 0) -> BISSEXTO

A Ordem de Precedência

O computador não lê a matemática de forma cega. Ele resolve primeiro os operadores mais "fortes".

1º
( )

Parênteses (A força suprema para quebrar as regras)

2º
* / %

Multiplicação, Divisão e Módulo (Da esquerda para a direita)

3º
+ -

Adição e Subtração

Ex: 5 + 3 * 2 resulta em 11, e não 16!

Bônus: Atribuições Compostas

Programadores são preguiçosos e gostam de digitar menos. O C oferece atalhos para atualizar o valor de uma variável usando a própria variável.

Em vez de escrever:

x = x + 5;

Você pode escrever:

x += 5;

O Atalho Supremo de +1 e -1:

Para somar exatamente 1, use: x++;

Para subtrair exatamente 1, use: x--;

Depurador: Vendo a Matemática Acontecer

(Use a Seta → do teclado para simular o processador resolvendo as linhas)

int a = 15;
int b = 4;
int div = a / b;
int resto = a % b;
a++; // a = a + 1
b += 2; // b = b + 2

O Código Completo: Calculadora

Veja como tudo funciona junto. O programa abaixo lê dois números e calcula tudo de uma vez.

#include <stdio.h>


int main() {


    int n1, n2;


    printf("Digite o valor de N1: ");


    scanf("%d", &n1);


    printf("Digite o valor de N2: ");


    scanf("%d", &n2);




    // Fazendo os cálculos


    int soma = n1 + n2;


    int sub = n1 - n2;


    int mult = n1 * n2;


    int divInteira = n1 / n2;


    int resto = n1 % n2;




    printf("\n--- Resultados ---\n");


    printf("Soma: %d\n", soma);


    printf("Divisao Inteira: %d\n", divInteira);


    printf("Resto (Modulo): %d\n", resto);




    return 0;


}
