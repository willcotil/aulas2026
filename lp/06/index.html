O Comando de Atribuição (=)

Na matemática, o sinal de igual (=) significa que os dois lados têm o mesmo valor. Na linguagem C, o igual significa "RECEBE".

A Regra de Ouro da Atribuição:

"O computador resolve TUDO o que está à DIREITA primeiro, e só no final guarda o resultado na variável que está à ESQUERDA."

// O computador calcula "10 + 5" primeiro (dá 15)
// E depois guarda o 15 na variável 'resultado'
int resultado = 10 + 5;

// Podemos usar variáveis já existentes do lado direito!
// Calcula "15 * 2" (dá 30) e guarda na variável 'dobro'
int dobro = resultado * 2;

Operadores Básicos

Os três primeiros operadores são exatamente iguais aos que aprendemos na escola.

Símbolo Operação Exemplo no Código
+ Adição soma = a + b;
- Subtração sub = 10 - 4;
* Multiplicação mult = 4 * 5;

Nota: O asterisco (*) substitui o clássico "x" ou "·" para evitar confusão com a letra x.

A Pegadinha da Divisão ( / )

A barra (/) faz a divisão, mas a linguagem C tem uma regra muito estrita sobre os tipos de dados que estão a ser divididos.

Divisão de Inteiros

Se você divide dois inteiros (int), o C corta as casas decimais. Ele não arredonda, ele simplesmente deita a vírgula fora!

10 / 4
-> Resultado: 2

Divisão Real

Para obter um resultado com decimais, pelo menos um dos números na conta precisa de ser do tipo float ou double (ter ponto).

10.0 / 4.0
-> Resultado: 2.5

A Estrela da Aula: O Módulo ( % )

O símbolo % não significa percentagem em programação! Ele significa "O RESTO DA DIVISÃO INTEIRA".

10 / 3

A divisão dá 3

(Pois 3 x 3 = 9)

🤔

10 % 3

O Resto é 1

(10 menos 9 = 1)

CUIDADO MÁXIMO: O operador Módulo (%) SÓ FUNCIONA com variáveis do tipo inteiro (int).

Para que serve o Módulo?

Saber o que sobra de uma divisão é um dos truques mais usados na programação lógica.

1. Descobrir se é PAR ou ÍMPAR

Qualquer número par dividido por 2 não tem resto (sobra 0). Se sobrar 1, é ímpar!

se (numero % 2 == 0) -> PAR

2. Encontrar Múltiplos

Para saber se um número é divisível por outro (ex: anos bissextos são divisíveis por 4).

se (ano % 4 == 0) -> BISSEXTO

A Ordem de Precedência

O computador não lê a matemática de forma cega. Ele resolve primeiro os operadores mais "fortes".

1º
( )

Parênteses (A força suprema para quebrar regras)

2º
* / %

Multiplicação, Divisão e Módulo (Da esquerda para a direita)

3º
+ -

Adição e Subtração

Ex: 5 + 3 * 2 resulta em 11, e não 16!

Atualizando Variáveis (Atribuição Composta)

É muito comum precisarmos somar ou alterar um valor baseando-se no que a variável já tem guardado. Em C, há um atalho para isso:

Forma Tradicional:

x = x + 5;

"Pegue o valor atual de x, some 5, e guarde de volta em x"

O Atalho (Forma Composta):

x += 5;

"Aumente o valor de x em 5"

Esse truque funciona com todos: -=, *=, /= e %=.

O Atalho Supremo: Incremento/Decremento

Programadores fazem tantas contagens de "1 em 1" (contar turnos, pontuação, tentativas) que inventaram um atalho exclusivo só para somar ou subtrair exatamente 1.

Incremento ( ++ )

É a exata mesma coisa que fazer x = x + 1;

vidas++;

Decremento ( -- )

É a exata mesma coisa que fazer x = x - 1;

vidas--;

Depurador: Vendo a Matemática Acontecer

(Use a Seta → do teclado para simular o processador a resolver as linhas)

int a = 15;
int b = 4;
int div = a / b;
int resto = a % b;
a++; // a = a + 1
b += 2; // b = b + 2

O Código Completo: Calculadora

Veja como tudo funciona junto. O programa abaixo lê dois números e calcula tudo de uma vez.

#include <stdio.h>
int main() {
    int n1, n2;
    printf("Digite o valor de N1: ");
    scanf("%d", &n1);
    printf("Digite o valor de N2: ");
    scanf("%d", &n2);

    // Fazendo os cálculos
    int soma = n1 + n2;
    int sub = n1 - n2;
    int mult = n1 * n2;
    int divInteira = n1 / n2;
    int resto = n1 % n2;

    printf("\n--- Resultados ---\n");
    printf("Soma: %d\n", soma);
    printf("Divisao Inteira: %d\n", divInteira);
    printf("Resto (Modulo): %d\n", resto);

    return 0;
}

Exercícios (Parte 1)

1. Teste de Mesa (Mental)

2. Divisão Quebrada

Um programador escreveu float media = 15 / 2; e ficou confuso porque o resultado impresso foi 7.000000 em vez de 7.5. Explique, com base na aula, por que isso aconteceu e como consertar a linha de código.

3. Média Ponderada (Código)

Crie um programa em C que calcule a média ponderada de duas notas. Peça ao utilizador a nota1 (peso 2) e a nota2 (peso 3).
A fórmula matemática é: Média = (nota1 * 2 + nota2 * 3) / 5.0. Exiba a média final.

4. A Caçamba de Maçãs (Código)

Uma fazenda embala maçãs em caixas que cabem exatamente 12 unidades. Crie um programa que pergunte ao utilizador o número total de maçãs colhidas. O programa deve calcular e exibir:
a) Quantas caixas completas foram formadas. (Dica: Divisão inteira)
b) Quantas maçãs sobraram fora da caixa. (Dica: Módulo)

5. Desafio: Inversor de Dois Dígitos

Crie um programa que pede ao utilizador um número inteiro de exatamente dois dígitos (ex: 74). O programa deve separar o número na dezena (7) e unidade (4) usando divisão inteira por 10 e módulo de 10. Por fim, exiba o número invertido (ex: 47).

Exercícios (Parte 2)

6. Semanas e Dias (Código)

Crie um programa que pergunte ao utilizador uma quantidade total de dias (ex: 45). O programa deve calcular e exibir quantas semanas completas existem nesses dias e quantos dias sobram.
Ex: 45 dias = 6 semanas e 3 dias. (Dica: Use / e %).

7. Calculadora de IMC (Código)

O Índice de Massa Corporal (IMC) é calculado dividindo o peso pela altura ao quadrado. Crie um programa que leia o peso (float) e a altura (float). Calcule e exiba o IMC.
Fórmula no código: imc = peso / (altura * altura);

9. Conversor de Temperatura (Código)

Crie um programa que leia uma temperatura em graus Fahrenheit (float) e a converta para Celsius. Exiba o resultado com 2 casas decimais.
Fórmula: C = (F - 32) / 1.8. Preste muita atenção aos parênteses!

10. Desafio do Multibanco (Caixa Eletrónico)

Um utilizador quer levantar dinheiro. O programa pede o valor exato a levantar (apenas inteiros, ex: 130). O multibanco só tem notas de 50, 20 e 10.
Usando apenas matemática (divisões inteiras e restos), calcule e exiba quantas notas de cada valor serão entregues para compor o total. Suponha que o valor digitado é sempre múltiplo de 10.
