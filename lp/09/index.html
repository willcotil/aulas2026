O Limite do IF Simples

Até agora, os nossos if faziam apenas uma pergunta por vez. Exemplo: "A nota é maior que 6?".

Mas no mundo real, muitas decisões dependem de várias exigências ao mesmo tempo.

Exemplo 1: Montanha-russa

Para entrar, a pessoa precisa ter a idade certa E a altura certa.

Idade >= 12 E Altura > 1.50

Exemplo 2: Isenção de Taxa

A pessoa não paga se for idosa OU se for estudante.

Idade > 65 OU Estudante == 1

Para resolver isso, usamos os Operadores Lógicos!

&&

O Operador AND

Conhecido como "E" Lógico. Ele é muito exigente. Para o resultado final ser Verdadeiro, TODAS as condições precisam ser verdadeiras.

"Para dirigir legalmente no Brasil, você precisa ser maior de idade E ter a carteira de habilitação."

// Os dois 'E' comerciais representam o AND em C
if (idade >= 18 && carteira == 1) {
    printf("Pode dirigir o carro!\n");
}

Se a pessoa tiver 20 anos, mas não tiver a carteira, o resultado final é Falso (0) e ela não dirige.

A Tabela Verdade do AND (&&)

O computador avalia cada lado e tenta juntá-los. Lembre-se: Verdadeiro = 1, Falso = 0.

Condição A Condição B Resultado (A && B)
0 (Falso) 0 (Falso) 0 (Falso)
0 (Falso) 1 (Verdadeiro) 0 (Falso)
1 (Verdadeiro) 0 (Falso) 0 (Falso)
1 (Verdadeiro) 1 (Verdadeiro) 1 (Verdadeiro)

Regra de Ouro do AND: Basta que UM seja Falso para que tudo desmorone e seja Falso.

||

O Operador OR

Conhecido como "OU" Lógico. Ele é muito flexível. Basta que APENAS UMA das condições seja verdadeira para que o resultado final seja Verdadeiro.

"O cliente tem frete grátis se comprar mais de R$ 200 OU se tiver um cupom VIP."

// As duas 'barras em pé' (Pipes) representam o OR em C
if (valor > 200.0 || cupom_vip == 1) {
    printf("Frete Gratis aplicado!\n");
}

Se o cliente tiver o cupom VIP, o frete é grátis mesmo que ele compre apenas 10 reais!

A Tabela Verdade do OR (||)

Avaliando ambos os lados com o operador OU. Verdadeiro = 1, Falso = 0.

Condição A Condição B Resultado (A || B)
0 (Falso) 0 (Falso) 0 (Falso)
0 (Falso) 1 (Verdadeiro) 1 (Verdadeiro)
1 (Verdadeiro) 0 (Falso) 1 (Verdadeiro)
1 (Verdadeiro) 1 (Verdadeiro) 1 (Verdadeiro)

Regra de Ouro do OR: Basta que UM seja Verdadeiro para que tudo seja Verdadeiro. Só será Falso se os dois forem Falsos.

!

O Operador NOT

Conhecido como "NÃO" Lógico. Ele é o do contra. Ele serve para INVERTER o estado de uma variável ou expressão.

!1 -> 0

O que era Verdadeiro vira Falso.

!0 -> 1

O que era Falso vira Verdadeiro.

// Ex: Se a porta NÃO estiver trancada, entre!
if (!porta_trancada) { ... }

Agrupando Condições (O Poder Máximo)

Podemos misturar && e || na mesma expressão! Para evitar que o computador se confunda, usamos parênteses para separar os blocos lógicos.

"Para pilotar, você precisa ser maior de idade E ter carteira, OU então ter uma autorização especial (mesmo sendo menor)."

if ( (idade >= 18 && carteira == 1) || autorizacao == 1 )

O C vai resolver os parênteses azuis primeiro (O bloco AND). Depois vai pegar a resposta desse bloco (0 ou 1) e comparar com o bloco verde usando o OR central.

(Pressione a Seta → para ver o computador transformar expressões em Verdadeiro ou Falso)

int a = 1; // Verdadeiro
int b = 0; // Falso

int res_or = (a || b);
int res_not = (!a);

O Código Completo: Login Simples

Usando o AND Lógico (&&) para garantir que tanto o usuário quanto a senha estejam corretos ao mesmo tempo.

#include <stdio.h>

int main() {
    int user_db = 999;
    int pass_db = 1234;
    
    int user_digitado, pass_digitada;

    printf("ID de Usuario: ");
    scanf("%d", &user_digitado);
    printf("Senha numerica: ");
    scanf("%d", &pass_digitada);

    // Exige que AMBOS os dados estejam corretos
    if (user_digitado == user_db && pass_digitada == pass_db) {
        printf("\n--> BEM-VINDO AO SISTEMA!\n");
    } else {
        printf("\n--> ACESSO NEGADO!\n");
    }

    return 0;
}

Resumo: Os 3 Operadores Lógicos

Um guia rápido para não esquecer como o computador combina informações.

&&

AND (E)

O Exigente

1 && 1 = 1

Só devolve Verdadeiro se TODOS os lados forem verdadeiros.

||

OR (OU)

O Flexível

0 || 1 = 1

Devolve Verdadeiro se PELO MENOS UM lado for verdadeiro.

!

NOT (NÃO)

O Inversor

!1 = 0

Inverte o valor lógico. O que era Verdadeiro vira Falso e vice-versa.

Exercícios (Teste de Mesa Mental)

Calcule o valor final (0 ou 1) que o computador atribuirá a cada expressão, sabendo que x=5 e y=10.

1. Teste Lógico Básico

int result = (x < 10) && (y == 10);

2. O Poder do OR

int result = (x > 10) || (y >= 10);

3. A Inversão (NOT)

int result = !(x == 5);

4. O Combo Mortal

Resolva primeiro os parênteses internos!

int result = ( (x == 5 && y < 5) || (y == 10) );

5. Consertando o Código

O aluno queria testar se uma variável nota estava entre 0 e 10. Ele escreveu: if (0 <= nota <= 10). Explique no seu arquivo de texto porque o C não compreende isso e reescreva a linha usando o operador lógico correto.
