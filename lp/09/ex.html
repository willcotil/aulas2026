⚔️ RPG Logic: O Duelo de Turno Único

O mestre da masmorra desafiou você a criar o sistema de combate do jogo. Como você ainda não aprendeu else if, seu desafio é usar Operadores Lógicos para manter o código organizado!

Objetivo: Determinar se o jogador acertouAtaque (1 ou 0) e se o jogador foiAtingido (1 ou 0).

📋 Regras de Negócio

  • Guerreiro (Classe 1): Acerta se dado > 12 E distancia < 3.
  • Arqueiro (Classe 2): Acerta se dado > 10 E distancia > 5.
  • Mago (Classe 3): Acerta se dado > 18. Ele ignora a distância.
  • Dano do Monstro: O monstro te atinge se !(classe == 3) E o dado for menor que 10.

⚠️ Atenção: Você pode usar else, mas não pode usar else if!

💻 Ponto de Partida

#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    int classe, dadoAtaque, distancia;
    int acertouAtaque = 0;
    int foiAtingido = 0;

    srand(time(NULL));
    dadoAtaque = (rand() % 20) + 1;

    printf("Escolha: 1-Guerreiro | 2-Arqueiro | 3-Mago: ");
    scanf("%d", &classe);
    printf("Distancia do monstro: ");
    scanf("%d", &distancia);

    printf("\n>>> Voce rolou o dado: %d <<<\n", dadoAtaque);

    // DICA: Em vez de fazer vários elses aninhados, 
    // tente usar: if (classe == 1 && ...)

    // ESCREVA SUA LOGICA AQUI
    


    // RESULTADOS FINAIS
    if (acertouAtaque == 1) {
        printf("Sucesso! O monstro foi atingido.\n");
    } else {
        printf("Voce errou o golpe...\n");
    }

    if (foiAtingido == 1) {
        printf("O monstro contra-atacou e te feriu!\n");
    } else {
        printf("Voce saiu ileso deste turno.\n");
    }

    return 0;
}
