Slide 03

O Problema: Menus Longos

Imagine que está a criar o menu principal de um jogo. O utilizador escolhe uma opção numérica. Com o que aprendemos até agora, faríamos assim:

menu_ruim.c
int opcao;
scanf("%d", &opcao);

if (opcao == 1) {
    printf("Novo Jogo");
} else if (opcao == 2) {
    printf("Carregar Jogo");
} else if (opcao == 3) {
    printf("Opcoes de Audio");
} else if (opcao == 4) {
    printf("Opcoes de Video");
} else if (opcao == 0) {
    printf("Sair do Jogo");
} else {
    printf("Opcao Invalida");
}

Funciona? Sim. É elegante? Nem um pouco. Repetir opcao == dez vezes torna o código chato de ler.

Fase 2: A Solução Switch

O Chaveamento Inteligente

Slide 04

A Solução: switch / case

A linguagem C (e muitas outras) possui uma estrutura específica para testar uma mesma variável contra vários valores exatos de forma muito mais limpa.

Como funciona?

Imagine o switch como uma linha de comboio (trem) com um mecanismo de desvio (um "switch" em inglês).

O valor da sua variável é a locomotiva. O switch avalia esse valor e direciona a locomotiva exatamente para a linha correspondente (o case correto), sem ter que perguntar a todas as linhas anteriores.

A Sintaxe
switch (variavel) {
    case 1:
        // faz algo se variavel for 1
        break;
    case 2:
        // faz algo se variavel for 2
        break;
    default:
        // faz algo se não for nenhum
        break;
}
Slide 05

O Menu Refeito com switch

Veja como o nosso menu do jogo fica muito mais organizado e fácil de ler quando utilizamos a estrutura de múltipla escolha.

menu_bom.c
int opcao;
scanf("%d", &opcao);

switch (opcao) {
    case 1:
        printf("Novo Jogo\n");
        break;
    case 2:
        printf("Carregar Jogo\n");
        break;
    case 3:
        printf("Opcoes de Jogo\n");
        break;
    default:
        printf("Opcao Invalida!\n");
        break;
}

Depurador: O Salto Exato do Switch

O switch não testa linha por linha como os vários if. Ele calcula o salto e vai diretamente para o case correto.

Slide 07

Os Dois Mandamentos do Switch

1. O Comando break

O switch é uma estrutura que "cai por ali abaixo". Quando ele encontra o case correto, ele entra nele e continua a executar todas as linhas que encontrar pela frente (inclusive os outros cases abaixo dele!).

Para evitar isso, SEMPRE colocamos um break; no fim de cada case. O break significa "Pára tudo e salta para fora do switch".

2. O Comando default

O default é exatamente igual ao else solitário do fim de uma cascata de ifs.

Ele funciona como a nossa Rede de Segurança. Se o utilizador digitar um número que não tem nenhum case correspondente, o código salta automaticamente para o default.

Slide 08

switch vs if / else: Quando usar qual?

O switch é muito elegante, mas tem limitações pesadas em C. Você não pode usá-lo sempre.

Limitações do Switch:

  1. O switch apenas testa a igualdade exata (como um ==). Você não pode testar "maior que" ou "menor que" (> ou <) dentro de um switch.
  2. O switch em C apenas funciona com números inteiros (int) ou letras (char). Não funciona com números decimais (float) nem com textos inteiros (strings/vetores).

Use switch para:

Menus de opções (1, 2, 3), categorias exatas (A, B, C), códigos de erro.

Use if / else para:

Faixas de valores (ex: notas maiores que 6), contas matemáticas, verificação de floats e condições compostas com &&.

Slide 09

Exemplo Supremo: Calculadora com char

Como o switch aceita letras (já que letras são números na tabela ASCII), ele é perfeito para identificar qual operador matemático o utilizador digitou!

calculadora.c
char operador;
float num1, num2;

printf("Digite a operacao (+, -, *, /): ");
scanf(" %c", &operador); // Espaço mágico antes do %c!
printf("Digite os dois numeros: ");
scanf("%f %f", &num1, &num2);

switch (operador) {
    case '+':
        printf("Resultado: %.2f\n", num1 + num2);
        break;
    case '-':
        printf("Resultado: %.2f\n", num1 - num2);
        break;
    case '*':
        printf("Resultado: %.2f\n", num1 * num2);
        break;
    case '/':
        // Dentro do case, podemos usar um IF normal!
        if (num2 == 0) {
            printf("Erro: Divisao por zero!\n");
        } else {
            printf("Resultado: %.2f\n", num1 / num2);
        }
        break;
    default:
        printf("Operador invalido!\n");
        break;
}
