Slide 03

O Problema da Previsibilidade

Até agora, os nossos programas eram exatamente iguais em todas as execuções. Se o código diz soma = 5 + 5, a resposta será sempre 10.

Para os Jogos, isso é péssimo!

Imagine jogar FIFA e o guarda-redes saltar sempre para o mesmo lado no penálti. Ou jogar Pokémon e o inimigo usar sempre o mesmo ataque na mesma ordem.

Precisamos de Aleatoriedade!

Para criar jogos divertidos ou sistemas de sorteio, precisamos que o computador nos dê um número surpresa de cada vez que rodarmos o programa.

Slide 04

A Função rand()

A linguagem C possui uma biblioteca padrão chamada <stdlib.h> que contém uma função chamada rand() (de random, que significa aleatório).

basico.c
#include <stdio.h>
#include <stdlib.h> // NECESSÁRIO para o rand()

int main() {
    int numero_sorteado = rand(); // Pede um número aleatório
    printf("O numero sorteado foi: %d\n", numero_sorteado);
    return 0;
}

O Problema do "Dia da Marmota"

Se compilar e rodar este código, ele pode imprimir 41. Mas se você rodar o programa de novo, amanhã, noutro PC... ele vai imprimir 41 de novo!

A função rand() é "pseudo-aleatória". Sem uma configuração inicial, ela usa sempre a mesma semente (fórmula) matemática e gera sempre a mesma sequência de números.

Fase 2: A Semente

Ensinando o PC a não se repetir

Slide 06

A Solução: srand() e o Relógio

Para resolver isso, precisamos dar uma "semente" inicial diferente para o gerador de números de cada vez que o programa abre. O que está sempre a mudar no computador? O relógio!

time(NULL)

Esta função devolve o número de segundos que se passaram desde o dia 1 de janeiro de 1970. Como o tempo nunca para, este número é sempre diferente a cada segundo.

srand()

Significa Seed Random (Plantar a Semente). Nós pegamos no número do relógio e colocamos dentro do srand() apenas uma vez no início do programa.

Slide 07

A Receita Completa de Aleatoriedade

Para gerar números realmente aleatórios e diferentes a cada execução, você deve sempre incluir as duas bibliotecas e plantar a semente no topo do seu main.

sorteio_real.c
#include <stdio.h>
#include <stdlib.h> // Para usar rand() e srand()
#include <time.h>   // Para usar time()

int main() {
    // 1. OBRIGATÓRIO: Planta a semente usando o relógio atual.
    // Faça isso APENAS UMA VEZ no programa todo!
    srand(time(NULL));

    // 2. Agora sim, podemos gerar números livremente.
    int a = rand();
    int b = rand();

    printf("Numero A: %d\n", a); // Ex: 15487
    printf("Numero B: %d\n", b); // Ex: 8712
    return 0;
}

Fase 3: O Limite

Domando os números gigantes

Slide 09

O Poder do Módulo ( % )

A função rand() devolve números gigantes (ex: 28473, 19284). Mas e se precisarmos de rolar um dado que só vai de 1 a 6?

Usamos o nosso velho amigo: O Módulo (Resto da Divisão)!

rand() % 6

Se dividirmos qualquer número gigante por 6, o resto da divisão nunca pode ser maior que 5!

Os restos possíveis são apenas: 0, 1, 2, 3, 4, 5.

Slide 10

Ajustando a Faixa (O +1 Mágico)

Como o módulo % 6 nos dá resultados de 0 a 5, e um dado normal vai de 1 a 6, nós apenas precisamos de somar 1 ao resultado final!

A Fórmula Universal

numero = (rand() % MAXIMO) + MINIMO;

  • Dado de 6 faces (1 a 6): (rand() % 6) + 1
  • Dado de 20 faces RPG (1 a 20): (rand() % 20) + 1
  • Moeda de Cara ou Coroa (0 ou 1): (rand() % 2) (Neste caso não soma 1, pois queremos 0 ou 1).

Desafios Práticos

Crie um arquivo .c para cada desafio, compile e teste no terminal. Não se esqueça de incluir as três bibliotecas e o srand(time(NULL));!

  1. O Cara ou Coroa:
    Crie um programa que simule o lançamento de uma moeda. Gere um número aleatório que pode ser apenas 0 ou 1. Utilize um if/else para imprimir "Cara!" se sair 0, e "Coroa!" se sair 1.
  2. Simulador de RPG (D20):
    Gere um número aleatório de 1 a 20 (simulando um dado D20).
    - Se tirar 20, imprima "ACERTO CRÍTICO! Dano máximo!".
    - Se tirar 1, imprima "ERRO CRÍTICO! Você acertou a si mesmo!".
    - Para qualquer outro valor, imprima "Você acertou o ataque com força [valor]".
  3. A Lotaria Simples:
    Peça ao utilizador para digitar um número da sorte de 1 a 10 (usando scanf). Em seguida, o computador sorteia um número aleatório de 1 a 10.
    Se o utilizador acertar, imprima "Parabéns! Você ganhou!". Senão, imprima "Que pena. O número sorteado foi X".
