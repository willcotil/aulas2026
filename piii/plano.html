Projeto Integrador II

Da documentação à prática. Desenvolvimento Multiplataforma Integrado.

Regra Obrigatória de Grupos

Os grupos devem ter estritamente 3 ou 4 membros. Grupos menores serão dissolvidos e integrados a outros. Grupos maiores deverão ceder membros.
Não haverá exceções. Organize-se na primeira semana.

Versionamento Obrigatório (GitHub)

Todo o desenvolvimento (código, banco de dados e documentação) deve ser mantido obrigatoriamente no GitHub. A avaliação considerará a frequência de commits de todos os membros do grupo. Projetos sem histórico de versão não serão aceitos.

Arquitetura do Projeto

O projeto iniciado no ano anterior deve agora ser implementado utilizando uma arquitetura centralizada. Recomenda-se fortemente o uso de uma API RESTful para conectar todas as pontas.

Web

Interface acessível via navegador consumindo a API.

Desktop

Aplicação local (Windows/Linux) consumindo a API.

Mobile

App nativo ou híbrido para dispositivos móveis.

Banco de Dados

Modelagem e implementação relacional ou não-relacional.
