Por que usar Planilhas?

Docs vs Sheets

No Google Docs (que vimos antes), o objetivo é escrever textos livres e relatórios.

O Google Sheets (Planilhas) é uma calculadora gigante unida com um banco de dados. O foco não é criar parágrafos, mas organizar dados para cálculos, listas e análises.

Tudo precisa estar em seu quadrado.

A Sopa de Letrinhas (Formatos)

Ao baixar ou enviar uma planilha, você encontrará diferentes extensões. O que elas significam?

.xls

O Antigo

Formato original do Excel (antes de 2007). Muito pesado e com limite pequeno de linhas. Evite usar hoje em dia.

.xlsx

O Padrão Moderno

Formato atual. Salva as cores, fórmulas, gráficos e suporta milhões de linhas de forma comprimida.

.csv

O Tradutor Universal

Comma Separated Values. É o formato mais importante para transportar dados entre sistemas diferentes.

O Poder do CSV

A Lógica da Simplicidade

Nem todo programa de software sabe ler planilhas do Excel. Mas TODO programa do mundo sabe ler um arquivo de Texto Simples.

O CSV tira todas as cores e fórmulas, e salva apenas os dados, separando as colunas com uma vírgula (ou ponto e vírgula).

Como o computador vê o arquivo .csv:

Nome,Idade,Setor

Ana,25,TI

Carlos,42,Vendas

Como a Planilha abre isso:

NomeIdadeSetor
Ana25TI
Carlos42Vendas

A intersecção exata entre a Coluna C e a Linha 4.

Referência Única

Cada quadradinho da planilha é chamado de Célula. Não existem duas células com o mesmo nome. A coordenada serve para dizermos ao computador exatamente onde buscar um dado para fazer uma conta.

O que cabe na Célula? (Dados Brutos)

1. Textos (Strings)

João Silva
Rua das Flores, 10

O computador não faz cálculos com texto.

2. Números (Values)

150
15.5
15/04/2026

Atenção: Datas são números disfarçados!

3. Fórmulas

=A1+B1
=SOMA(C1:C10)

A mágica. Toda célula que começa com o sinal de = (Igual) é uma ordem de cálculo.

O Sistema Alinha Sozinho

Textos grudam na esquerda para lermos como num livro.

Números grudam na direita para que as vírgulas fiquem perfeitamente alinhadas (unidade embaixo de unidade), facilitando a soma visual pelo cérebro humano.

Formatando Valores (A Roupa do Número)

O Erro Iniciante ❌

Se você digitar "R$ 1500,00" usando as letras do teclado, o sistema vai achar que isso é um TEXTO. Quando tentar somar com outro valor, vai dar um erro: #VALOR!

Você deve digitar APENAS o número puro: 1500

Como vestir a Roupa do Número?

1. Selecione a célula.

2. Vá na barra e clique no botão 123 ▼ (ou o botão R$ direto na barra).

Você digita:

10

➜

Aparece na tela:

R$ 10,00

Como o computador lê isso?

Para a máquina, não é um desenho. Se o quadrado está marcado, a célula vale VERDADEIRO. Se vazio, vale FALSO.

Evitando Erros: Lista Suspensa 🔽

Se você deixar as pessoas digitarem o status livremente, uma vai escrever "Aprovado", a outra "Aprovdo". Sua tabela vai virar um caos.

A solução é criar um "Menu Suspenso" (Dropdown). O usuário só pode escolher as opções que você permitir!

Caminho: Inserir > Menu Suspenso

A Barra de Ferramentas (1/4): Controle

Vamos fatiar a barra superior para o cérebro entender. Esta primeira parte controla o sistema e a aparência matemática da célula.

Controles do Sistema

  • 🔍 Menus
    Busca inteligente. Não achou um botão? Digite aqui e ele faz para você.
  • Desfazer/Refazer: Volte no tempo se errar (Ctrl+Z).
  • Imprimir: Prepara a planilha para o papel.
  • Pintar Formatação: Copia o visual de uma célula e cola em outra.

Identidade Numérica

  • R$
    Moeda: Botão rápido para transformar dígito '15' em 'R$ 15,00'.
  • %
    Porcentagem: Converte decimais (ex: 0,15 vira 15%).
  • .0← / .00→
    Casas Decimais: Aumenta ou esconde os zeros depois da vírgula.
  • 123 ▼
    Mais Formatos: Menu para transformar números em Datas (25/12), Horas (14:00) ou Texto Simples.

A Barra de Ferramentas (2/4): Estilo Visual

Este bloco inteiro cuida apenas da estética: a tinta e o formato das letras dentro dos quadrados.

Design do Texto

  • Padrão... ▼
    Fonte: Muda a "caligrafia" (Arial, Roboto).
  • - 10 +
    Tamanho: Aumenta ou diminui a letra.
  • B
    Negrito: Letra gorda. Ideal para cabeçalhos.
  • I
    Itálico: Letra deitada. Para notas.
  • S
    Tachado: Risco no meio. Sinaliza tarefa cancelada.

Tinta e Pigmento

  • A
    Cor do Texto: Muda a cor da tinta da caneta. O padrão é preto.
  • Cor de Preenchimento: Derrama tinta no fundo do quadrado. A letra não muda. Essencial para criar alertas.

A Barra de Ferramentas (3/4): Estrutura

Aqui manipulamos a arquitetura física da grade e onde o texto se encosta nas paredes.

As Paredes (Físico)

  • Bordas: Pinta de preto as linhas invisíveis da grade. Necessário para a tabela aparecer na impressão.
  • Mesclar Células: Quebra as paredes entre duas ou mais células, fundindo-as em uma sala gigante (ótimo para Títulos).

Posicionamento Lógico

  • Alinhamento Horizontal: Empurra o texto para a parede Esquerda, Centro ou Direita.
  • Alinhamento Vertical: Cola o texto no Teto, no Meio ou no Chão da célula.
  • Ajuste de Texto: Se o texto for enorme, força ele a pular de linha e crescer a célula para baixo.

A Barra de Ferramentas (4/4): Inteligência

Estes botões são as ferramentas avançadas. Eles transformam células mortas em dados interativos e cálculos.

Inserções Extras

  • Link: Transforma a palavra num botão que abre um site externo.
  • Comentário: Prende um "post-it" amarelo na célula para colegas de trabalho lerem (Nuvem).
  • Gráfico: Transforma uma coluna de números em um gráfico de fatias de pizza instantâneo.

Operações de Dados

  • Criar Filtro: Oculta o que você não quer ver (Ex: Mostre apenas produtos "Aprovados").
  • Σ ▼
    Funções: O botão mais poderoso. Abre o catálogo com as fórmulas matemáticas (SOMA, MÉDIA, PROCV).

Mão na Massa 💻

AÇÃO: Seu primeiro banco de dados

  1. Abra o Google Sheets (Planilhas) e crie um arquivo Em Branco.
  2. Na linha 1, crie o Cabeçalho: Produto | Qtd | Preço Unitário | Pronto?
  3. Coloque o Cabeçalho em Negrito e centralizado.
  4. Adicione 4 produtos fictícios de tecnologia abaixo.
  5. Preencha a quantidade e o preço puramente numérico (ex: 150,50).
  6. Selecione toda a coluna de "Preço", vá na barra e clique no botão R$.
  7. Selecione a coluna "Pronto?", vá em Inserir > Caixa de Seleção.
  8. Selecione a tabela inteira e aplique Formatar > Cores Alternadas.

Linhas (1,2,3) x Colunas (A,B,C).

Cores Alternadas guiam a leitura humana.
