Por que Electron?

Vocês aprenderam C# com Windows Forms. Funciona, mas é limitado ao Windows e tem um visual "datado".

O Electron permite usar tecnologias Web (HTML, CSS, JS) para criar programas Desktop.

  • Multiplataforma: O mesmo código roda em Windows, Linux e Mac.
  • Visual: Interfaces modernas e fluidas (CSS).
  • Mercado: As maiores empresas do mundo usam.

06

O Javascript e o Node.js

Originalmente, o Javascript só rodava dentro do Navegador (Chrome, Firefox). Ele foi feito para a Web.

Para criar programas Desktop (Electron) ou Back-end, precisamos tirar ele do navegador. Aí entra o Node.js.

No Navegador

Controla a página, manipula HTML, animações.

No Node.js (Nosso caso)

Acessa arquivos, banco de dados, sistema operacional (Igual C).

O Node.js será o nosso "motor".

Variáveis: O Fim da Tipagem Forte

Em C, você define o tipo. Em JS, você define o comportamento.

  • const: Valor constante (não muda). Use por padrão.
  • let: Valor variável (pode mudar).
  • var: Antigo. NÃO USE (Vaza escopo).
// C
int idade = 18;
const float PI = 3.14;

// JS
let idade = 18;
const PI = 3.14;

Tipagem Dinâmica

Em JS, a variável não tem tipo. O valor tem tipo.

let caixa = 10;      // Agora é Number
console.log(typeof caixa); 

caixa = "Sapatos";   // Agora é String
console.log(typeof caixa);

Isso é flexível, mas perigoso. Cuidado ao misturar tipos.

15

Template Strings: A Mágica do `${}`

Esqueça o printf("Idade: %d", idade) ou a concatenação com `+`.

No JS, usamos Crasés (Backticks): ` `.

Como funciona a Substituição?

Quando o JS encontra ${...} dentro de uma string com crase, ele:

  1. Pega o que está dentro das chaves (pode ser variável ou cálculo).
  2. Calcula o valor.
  3. Transforma em texto e "cola" no lugar das chaves.
let nome = "Ana";
// O JS troca ${nome} por "Ana"
console.log(`Olá, ${nome}!`);

Condicionais: If, Else If, Else

A estrutura básica é idêntica ao C, mas podemos omitir chaves se for uma linha só.

let nota = 8;

if (nota >= 9) {
    console.log("Excelente");
} else if (nota >= 6) {
    console.log("Aprovado");
} else {
    console.log("Reprovado");
}

// Uma linha (sem chaves)
if (nota == 10) console.log("Parabéns!");

20

Decisão em uma linha: Operador Ternário

Existe uma forma de fazer um if/else simples em uma única linha. É o operador ? :.

Sintaxe: condição ? valor_se_verdade : valor_se_falso

Modo If/Else

let status;
if (idade >= 18) {
    status = "Maior";
} else {
    status = "Menor";
}

Modo Ternário

const status = idade >= 18 ? "Maior" : "Menor";
// Lê-se: Idade >= 18? 
// Se sim, "Maior". Se não, "Menor".
21

Switch Case

Idêntico ao C. Útil quando você tem valores exatos para testar (menus, opções).

const cor = "vermelho";

switch (cor) {
    case "verde":
        console.log("Siga");
        break;
    case "vermelho":
        console.log("Pare");
        break;
    default:
        console.log("Atenção");
}

Lista de Exercícios (Hard Mode)

Agora é com vocês. Resolvam usando apenas o que vimos hoje.

  1. Conversor de Temperatura: Declare uma variável celsius. Calcule Fahrenheit (`F = C * 1.8 + 32`) e Kelvin (`K = C + 273.15`). Exiba ambos.
  2. Calculadora de IMC: Declare peso e altura. Calcule o IMC (`peso / (altura * altura)`). Imprima o valor com 2 casas decimais.
  3. Verificador de Par/Ímpar: Declare um número. Use o operador `%` e um if/else para dizer se é par ou ímpar.
  4. Maior de 3: Declare 3 números diferentes. Use if/else aninhados para descobrir e imprimir o maior deles.
  5. Calculadora Switch: Declare n1, n2 e operacao (+, -, *, /). Use switch para calcular e imprimir o resultado.
  6. Bissexto: Declare ano. Verifique se é bissexto (Divisível por 4. Se divisível por 100, tem que ser por 400 também). Lógica pura!
  7. Conversor de Tempo: Declare uma variável minutos (ex: 130). Converta para o formato "X horas e Y minutos" (use divisão e resto).
  8. Bhaskara: Declare a, b, c. Calcule o Delta. Se negativo, imprima "Sem raízes". Se positivo, calcule e imprima as raízes.
  9. Ternário de Login: Declare logado = true ou false. Use APENAS o operador ternário para imprimir "Bem-vindo" ou "Faça Login".
  10. O Triângulo: Declare 3 lados. Primeiro verifique se formam um triângulo (soma de 2 lados > 3º lado). Se sim, use ifs aninhados para dizer se é Equilátero (3 iguais), Isósceles (2 iguais) ou Escaleno (todos diferentes).
