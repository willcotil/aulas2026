Parte 0: JS no Navegador

Saindo do Terminal

03

Do Node para o Navegador

Até agora, rodávamos node arquivo.js no terminal. O Node é ótimo para servidores e automação.

Porém, para criar interfaces visuais (Web ou Electron), precisamos que o Navegador execute nosso código.

Node.js

Sem interface gráfica (Apenas texto).

Acesso a arquivos do PC.

Navegador (Browser)

Interface Visual Rica (DOM).

Segurança restrita (Sem acesso direto a arquivos).

04

A Tag <script>

É ela que conecta o HTML ao Javascript.

<!-- index.html -->
<body>
  <h1>Minha Página</h1>

  <!-- Recomendado: Antes de fechar o body -->
  <script src="script.js"></script>
</body>

Importante: O JS deve carregar POR ÚLTIMO para garantir que os elementos da tela já existam.

05

Onde está o console.log?

No navegador, não temos o terminal do VSCode. Temos o Console do Desenvolvedor.

Como acessar:

  1. Abra seu arquivo HTML no Chrome.
  2. Pressione F12 (ou botão direito > Inspecionar).
  3. Clique na aba Console.

É lá que veremos erros e mensagens.

Exercício 0: Hello Browser

  1. Crie um arquivo index.html (digite ! e tab no VSCode).

  3. No HTML, adicione <script src="app.js"></script> antes de fechar o body.
  4. Abra o HTML no navegador, aperte F12 e veja a mensagem no Console.

Parte 1: O DOM

Document Object Model

08

O que é o DOM?

Quando o navegador carrega uma página, ele cria uma Árvore de Objetos que representa o HTML.

O Javascript não vê "texto" HTML, ele vê essa árvore. Cada tag (div, p, button) vira um Objeto no JS.

HTML (Texto)

<div>
  <h1>Olá</h1>
</div>

DOM (Memória)

document (Raiz)

└─ body

   └─ div

      └─ h1 ("Olá")

09

Selecionando Elementos

Para mexer em algo, primeiro precisamos "pegar" esse algo. Antigamente usávamos getElementById, mas hoje usamos o poderoso querySelector.

// Pega pelo ID (igual CSS)
const titulo = document.querySelector("#meuTitulo");

// Pega pela Classe
const botoes = document.querySelector(".btn-salvar");

// Pega pela Tag
const paragrafo = document.querySelector("p");

Se houver vários, ele pega apenas o primeiro!

Parte 2: Manipulação

Alterando a página dinamicamente

11

Mudando o Conteúdo

Depois de selecionar, podemos mudar o que está escrito.

const titulo = document.querySelector("h1");

// Muda apenas o texto
titulo.innerText = "Novo Título";

// Muda o HTML (permite tags)
titulo.innerHTML = "Título <i>Itálico</i>";

Teste ao Vivo

Texto Original

12

Mudando o Estilo (CSS)

Podemos acessar qualquer propriedade CSS via objeto style.

Nota: Propriedades com hífen viram CamelCase (ex: background-color vira backgroundColor).

const caixa = document.querySelector("#minhaCaixa");

caixa.style.backgroundColor = "red";
caixa.style.fontSize = "20px";
caixa.style.display = "none"; // Esconde
13

Clique nos botões para manipular o estilo da caixa via JS.

Caixa

Bastidores: Como funciona?

Aqui está o código Javascript que fez a mágica acontecer no slide anterior.

Função Pintar

function paint(cor) {
    const box = document.querySelector("#paintBox");
    box.style.backgroundColor = cor;
}

Função Círculo (Toggle)

function toggleCircle() {
    const box = document.querySelector("#paintBox");
    
    if (box.style.borderRadius === "50%") {
        box.style.borderRadius = "10px";
    } else {
        box.style.borderRadius = "50%";
    }
}

Parte 3: Eventos

O Fim da Execução Linear

16

Programação Orientada a Eventos

Em C, o programa começa no main() e vai linha a linha até o fim.

No JS (Front-end), o código carrega e fica esperando algo acontecer (um clique, uma tecla, o mouse mexer).

Nós criamos "Escutadores" (Listeners).

17

addEventListener

A função mais importante do DOM. Ela liga um evento a uma função.

const botao = document.querySelector("#btnSalvar");

// Quando 'click' acontecer, execute a função
botao.addEventListener("click", () => {
    alert("Salvando...");
    console.log("Clicou!");
});

Note o uso de uma Arrow Function como callback!

18

Lendo Inputs (Caixas de Texto)

Para pegar o que o usuário digitou, não usamos innerText, usamos value.

const campo = document.querySelector("#nome");
const valor = campo.value;

Evento 'input': Dispara a cada letra digitada.

campo.addEventListener("input", () => {
    console.log("Digitando: " + campo.value);
});
19

Prática: O Espelho

Tudo o que você digita no input aparece no texto abaixo instantaneamente.

Resultado:

...
input.addEventListener("input", () => {
    output.innerText = input.value;
});

Parte 4: Criando Elementos

HTML Dinâmico

21

Criando e Adicionando

Podemos criar tags novas que não existiam no HTML original.

  1. Criar: document.createElement("tag")
  2. Preencher: Mudar innerText ou estilo.
  3. Adicionar: pai.appendChild(filho)
// 1. Cria
const novoItem = document.createElement("li");

// 2. Preenche
novoItem.innerText = "Comprar Leite";

// 3. Adiciona na lista existente (ul)
document.querySelector("ul").appendChild(novoItem);
