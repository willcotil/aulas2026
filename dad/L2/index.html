1. To-Do List Espartano

Iniciante
Crie um aplicativo de tarefas que use o Node.js como "cérebro" para guardar as informações enquanto o aplicativo estiver aberto (em memória).
  • Crie a estrutura profissional de pastas (src/main e src/renderer).
  • No HTML, construa um layout com Bootstrap: Um formulário no topo e uma área para as tarefas embaixo. O formulário deve ter um input para o nome_tarefa e um select para a prioridade (Baixa, Média, Alta).
  • No JS do Front-end, intercepte o submit do formulário, use o new FormData() para capturar os dados, e envie o objeto para o Back-end via IPC (invoke).
  • No Node.js (Main), crie um Array global vazio. Sempre que o canal IPC for acionado, receba a nova tarefa, adicione um ID único a ela (pode ser o tamanho do array + 1), dê um .push() no array e retorne a lista inteira atualizada.
  • Quando o Front-end receber a lista de volta, use .map() para renderizar Cards do Bootstrap na tela. Destaque tarefas de "Alta" prioridade com uma cor diferente!

2. Gestão de Biblioteca (JSON)

Intermediário
Um sistema para visualizar e emprestar livros. O Back-end agora não inicia vazio; ele carrega dados de um arquivo externo!
livros.json (Crie na raiz do projeto) 
[
  { "id": 1, "titulo": "O Senhor dos Anéis", "autor": "J.R.R. Tolkien", "ano": 1954, "disponivel": true },
  { "id": 2, "titulo": "1984", "autor": "George Orwell", "ano": 1949, "disponivel": false },
  { "id": 3, "titulo": "Dom Quixote", "autor": "Miguel de Cervantes", "ano": 1605, "disponivel": true }
]
  • No Main, use require('../../livros.json') para carregar os livros para dentro de uma variável let. Crie um ipcMain.handle('get-livros') que apenas retorna essa variável para o Front-end renderizar a tela inicial.
  • No Front-end, além de desenhar os livros, crie uma Barra de Busca Dinâmica. Ao digitar qualquer coisa (evento input), filtre a lista e re-renderize os cards.
  • Ação no Backend: Em cada card de livro gerado no HTML, adicione um botão "Emprestar". Ao clicar neste botão, o Front-end deve enviar o id do livro via IPC para o Node.js.
  • No Node.js, crie um ouvinte IPC que recebe esse ID, encontra o livro correspondente no array, altera a propriedade disponivel de true para false, e devolve o array inteiro atualizado!
  • Regra Visual: Se o livro estiver com disponivel: false, o card dele no HTML deve ficar cinza (ou opaco) e o botão "Emprestar" deve sumir!

3. Tracker Financeiro Pro

Avançado + IA
Um dashboard para controle de despesas. O nível sobe: você não apenas lerá o JSON, mas fará com que o sistema o atualize de verdade (persistência de dados) e gerará gráficos interativos.
transacoes.json (Crie na raiz) 
[
  { "id": 1, "descricao": "Salário", "valor": 4500, "tipo": "receita", "data": "2026-04-05" },
  { "id": 2, "descricao": "Aluguel", "valor": 1200, "tipo": "despesa", "data": "2026-04-10" },
  { "id": 3, "descricao": "Supermercado", "valor": 600, "tipo": "despesa", "data": "2026-04-12" }
]
  • Construa uma interface Dashboard usando o sistema de Grid do Bootstrap (col-md-4, etc). O topo deve exibir cards de resumo calculados no JS: "Total Receitas", "Total Despesas" e "Saldo Atual".
  • Crie um formulário complexo para registrar novas transações (Descrição, Valor Numérico, Tipo [Radio Button] e Data [Input Date]). Capture tudo com FormData e envie ao Main.
  • Validação no Main: O Node.js deve rejeitar transações cujo valor seja menor ou igual a 0. Se passar, adicione ao Array em memória.

Missão Crítica: Engenharia de Prompts

Use o ChatGPT, Gemini ou Claude para resolver os dois bloqueios técnicos do seu sistema. Em vez de fornecermos o texto pronto, o seu desafio é formular a pergunta técnica perfeita.

Desafio A (A Persistência): Nós usamos require() para ler arquivos, mas como SALVAMOS o array modificado de volta no arquivo físico para os dados não sumirem quando o app fechar?

Formule um prompt técnico exigindo que a IA explique o uso do módulo nativo fs (File System) do Node.js para salvar/sobrescrever um array dentro de um arquivo .json.

Desafio B (O Gráfico): Um Dashboard financeiro não é nada sem gráficos interativos.

Formule um prompt pedindo à IA um exemplo prático de como importar a biblioteca Chart.js (via CDN) e como gerar um Gráfico de Pizza (Doughnut) baseado nos totais de receitas e despesas que calculou no Javascript.
