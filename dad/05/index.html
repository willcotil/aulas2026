Ex 01

1. Gerador de Lista Em Aula

O dado mais básico que recebemos de um backend é um array de textos. Vamos criar uma lista no HTML.

const alunos = ["Ana", "Bruno", "Carlos", "Daniela"];

Requisitos

  • Crie uma <ul id="lista"></ul> no HTML.
  • Crie uma Arrow Function chamada renderizarLista().
  • Use map() para transformar cada nome numa string "<li>Nome</li>".
  • Una o array resultante (pesquise sobre o método .join("")) e insira no innerHTML da ul.

Resultado Visual

  • Ana
  • Bruno
  • Carlos
  • Daniela
Ex 02

2. Filtro de Produtos Em Aula

Combinando filter() e map() em Objetos.

const produtos = [
  { nome: "Mouse", preco: 50 },
  { nome: "Teclado", preco: 150 },
  { nome: "Monitor", preco: 900 }
];

Requisitos

  • Crie um botão "Filtrar > R$100" no HTML e uma div vazia.
  • Adicione um evento de clique no botão (usando arrow function).
  • No clique, faça um filter() para pegar produtos com preço maior que 100.
  • Depois aplique map() para gerar parágrafos exibindo o Nome e o Preço, e jogue na div.

Resultado Visual

Teclado - R$ 150

Monitor - R$ 900

Ex 03

3. Totalizador de Carrinho Em Aula

Cálculos baseados em Arrays de Objetos.

const carrinho = [
  { item: "Livro", qtd: 2, valor: 30 },
  { item: "Caneta", qtd: 5, valor: 2 }
];

Requisitos

  • Use uma Arrow Function chamada calcularTotal().
  • Crie uma variável `total = 0`.
  • Use forEach() ou map() para multiplicar a qtd pelo valor de cada item e somar ao `total`.
  • Exiba o resultado num <h2> no HTML (Ex: "Total: R$ 70,00").

Resultado Visual

Total: R$ 70,00

Ex 04

4. Renderização Condicional Em Aula

O HTML deve mudar dependendo do valor booleano do objeto.

const tarefas = [
  { desc: "Estudar JS", feita: true },
  { desc: "Fazer exercícios", feita: false }
];

Requisitos

  • Crie uma lista (ul).
  • Use map() para gerar os <li>.
  • Dentro do map, use um Operador Ternário: Se feita for true, adicione uma classe CSS que deixa o texto verde e riscado. Senão, classe normal.

Resultado Visual

  • Estudar JS
  • Fazer exercícios
Ex 05

5. Busca Dinâmica (Search) Em Aula

Filtrando em tempo real conforme o usuário digita.

const contatos = [
  { nome: "João Silva", tel: "123" },
  { nome: "Maria Sousa", tel: "456" }
];

Requisitos

  • Crie um <input type="text" id="busca">.
  • Adicione um evento input ao campo.
  • No callback (arrow function), pegue o valor digitado.
  • Use filter() para achar os nomes que contêm (includes()) o texto digitado e mostre o resultado abaixo.

Resultado Visual

Maria Sousa - 456
